# 🚪 Exit the Echo - Digital Literacy Escape Room Game

A React-based educational game that teaches critical thinking and digital literacy skills through an immersive escape room experience.

## 🎮 Game Overview

**Exit the Echo** is an interactive web game that challenges players to escape from "The Echo Chamber" - a digital prison of misinformation. Players must navigate through 5 increasingly difficult levels, each testing different aspects of digital literacy:

### 🎯 Levels

1. **Level 1: Fake Feed** - Identify fake social media posts
2. **Level 2: Bot Army** - Spot automated bot accounts in a chatroom
3. **Level 3: Deepfake Dungeon** - Detect AI-generated or manipulated media
4. **Level 4: Bias Mirror** - Classify news headlines by media bias
5. **Level 5: Viral Hoax** - Find verified sources among competing claims

## 🚀 Features

- **Immersive Storyline**: Engaging narrative that guides players through the escape room
- **Interactive Learning**: Hands-on experience with real-world digital literacy challenges
- **Educational Content**: Built-in tips and explanations for each skill
- **Beautiful UI**: Modern, cyberpunk-themed design with smooth animations
- **Responsive Design**: Works on desktop and mobile devices
- **Progress Tracking**: Score system and level completion tracking

## 🛠️ Technology Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS with custom animations
- **Build Tool**: Create React App
- **Language**: TypeScript for type safety

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd exit-the-echo
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Open your browser** and navigate to `http://localhost:3000`

## 🎯 How to Play

1. **Start the Game**: Click "ENTER THE CHAMBER" on the intro screen
2. **Complete Each Level**: Follow the instructions for each challenge
3. **Learn from Feedback**: Read the explanations and tips provided
4. **Progress Through Levels**: Successfully complete each level to advance
5. **Escape**: Complete all 5 levels to escape The Echo Chamber

## 🎨 Game Mechanics

### Level 1: Fake Feed
- Click on posts that appear to be fake or contain misinformation
- Look for sensationalist language, missing sources, and suspicious claims
- Identify 3 fake posts to proceed

### Level 2: Bot Army
- Identify automated bot accounts in a chatroom
- Look for repetitive messages, promotional language, and unnatural patterns
- Select 3 bot accounts to continue

### Level 3: Deepfake Dungeon
- Analyze media clips for signs of AI manipulation
- Check for unnatural movements, inconsistent lighting, and artificial elements
- Identify the 1 deepfake among real content

### Level 4: Bias Mirror
- Classify news headlines by their political bias
- Learn to recognize left-leaning, right-leaning, and neutral reporting
- Correctly categorize all 3 headlines

### Level 5: Viral Hoax
- Cross-check multiple sources for a viral story
- Evaluate source credibility and identify verified information
- Find the 1 trustworthy source among fake news sites

## 🎓 Educational Value

This game teaches essential digital literacy skills:

- **Critical Thinking**: Question information and sources
- **Source Verification**: Check credibility and fact-check claims
- **Media Bias Recognition**: Identify political and commercial biases
- **Deepfake Detection**: Spot AI-generated or manipulated content
- **Bot Identification**: Recognize automated accounts and spam

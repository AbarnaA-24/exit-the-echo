# 🚀 How to Run "Exit the Echo"


1. **Install dependencies**: Run `npm install` in the project directory
2. **Start the development server**: Run `npm start`
3. **Open your browser**: Navigate to `http://localhost:3000`
4. **Play the game**: All 5 levels are now available!

**✅ The project is now working!** The development server should be running at `http://localhost:3000`



## 🎮 How to Play

1. **Start**: Click "ENTER THE CHAMBER" on the intro screen
2. **Level 1**: Identify 3 fake social media posts
3. **Level 2**: Spot 3 bot accounts in a chatroom
4. **Level 3**: Detect 1 deepfake among media clips
5. **Level 4**: Classify 3 headlines by media bias
6. **Level 5**: Find the verified source among competing claims
7. **Victory**: See your final score and skills acquired
